### CONFMA API DJANGO

# Requisitos

- Python 3.7+
- Django 3.0.7+

# Instalacion

1. Ejecutar el virtual enviroment
   - source bin/activate
2. Entrar a la carpeta del proyecto
   - instalar las librerias
     - pip install -r requirements.txt
3. ejercutar el proyecto
   - python manage.py runserver
